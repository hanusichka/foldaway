Foldaway is a travel planning application that brings together the tools people usually have to juggle separately — place lists, notes, maps, and place-search services — into a single, unified workspace.

Foldaway lets users register or log in, then create and manage their own trips. Within each trip, users can build place lists, add and edit locations, and track which ones have already been visited. All the essential trip data lives in one place, so there’s no need to switch between several apps while planning a journey.
